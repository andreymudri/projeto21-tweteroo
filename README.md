

## Description
- Tweeteroo is an API for a clone-line of Twitter, this being the first project i developed on NestJS.
- Keep in mind it does not have data persistency(database). every time you restart the server it's messages will be wiped out.


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
